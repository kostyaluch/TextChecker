# ============================================================================
# Єдиний Словник Правил / Unified Rules Catalog
# ============================================================================
# 
# Формат: ТІППРАВИЛА|параметр1|параметр2|...
#
# Типи правил:
# 1. ERROR_REPLACEMENT - помилки машинного перекладу (RU → UA)
#    Формат: ERROR_REPLACEMENT|російський_корінь|українські_корені_через_кому
#    Дія: якщо в RU є ru_stem, а в UA є ua_stem → фіксує помилку
#
# 2. IGNORE - винятки (правильні фрази, які не є помилками)
#    Формат: IGNORE|фраза_українською
#    Дія: якщо фраза присутня в реченні → ігнорує помилку
#
# 3. AUTO_REMOVE - автоматичне видалення (чорний список)
#    Формат: AUTO_REMOVE|термін
#    Дія: параграфи/речення з цим терміном вирізаються з форматованого тексту
#
# 4. FLAG_ONLY - тільки позначення (без видалення)
#    Формат: FLAG_ONLY|термін
#    Дія: термін фіксується в помилках/звіті, але не видаляється автоматично
#
# ============================================================================

# --- ERROR_REPLACEMENT: Помилки машинного перекладу ---

# Помилка: «Питание» (електричне) перекладається як «живлення» (наприклад, блок живлення). 
# Чому виникає: Перекладач обирає біологічне значення слова — «харчування» (прийом їжі).
ERROR_REPLACEMENT|питани|харчуванн

# Помилка: Слово «лук» в описі ігрової периферії або ноутбуків (як ігрова зброя) має залишатися «луком».
# Чому виникає: Перекладач розпізнає це слово як овоч і видає «цибуля».
ERROR_REPLACEMENT|лук|цибул

# Помилка: «Разрешение экрана» або камери — це «роздільна здатність».
# Чому виникає: Програма перекладає слово в юридичному сенсі — як «дозвіл» на виконання якоїсь дії.
ERROR_REPLACEMENT|разрешени|дозвіл

# Помилка: Технічний термін «отношение сигнал/шум» або «соотношение сторон» — це «відношення» або «співвідношення».
# Чому виникає: Перекладач бере емоційне значення — «ставлення» (наприклад, ставлення людини до роботи).
ERROR_REPLACEMENT|отношени|ставленн

# Помилка: «Шаг пикселя» українською — «крок пікселя».
# Чому виникає: В українській мові слово «шаг» означає лише дрібну старовинну монету. Фізична відстань або дія ногами — це завжди «крок».
ERROR_REPLACEMENT|шаг|шаг

# Помилка: «Рабочее напряжение» — це «робоча напруга».
# Чому виникає: «Напруження» означає емоційний або фізичний стан (напруження м'язів, нервове напруження).
ERROR_REPLACEMENT|напряжени|напруженн

# Помилка: «Сетевой адаптер» або «локальная сеть» — це «мережа».
# Чому виникає: Перекладач плутає цифрову мережу з фізичним виробом із ниток чи дроту (москітна сітка).
ERROR_REPLACEMENT|сеть|сітка

# Помилка: «Проводной/беспроводной интерфейс» українською буде «дротовий / бездротовий» (від слова дріт).
# Чому виникає: Звичайна калька (суржик) з російської мови.
ERROR_REPLACEMENT|проводн|проводн
ERROR_REPLACEMENT|беспроводн|безпроводн

# Помилка: «Емкость аккумулятора» — це «ємність» (наприклад, ємність батареї Apple iPad Pro).
# Чому виникає: Слово «ємкість» є поширеним суржиком.
ERROR_REPLACEMENT|емкост|ємкіст

# Помилка: «Расход энергии» має перекладатися як «витрата».
# Чому виникає: «Розхід» — це русизм і розмовний покруч, якого уникають у технічній документації.
ERROR_REPLACEMENT|расход|розхід

# Помилка: «Руководство пользователя» — це «посібник користувача» або «інструкція» (наприклад, посібник для Samsung Galaxy Tab A9).
# Чому виникає: Перекладач обирає адміністративне значення — «керівництво» (група людей, яка керує компанією).
ERROR_REPLACEMENT|руководств|керівництв

# Помилка: «Мобильное приложение» — це «застосунок» або «додаток».
# Чому виникає: Перекладається як фізична дія — «прикладання» (наприклад, зусиль або подорожника до рани).
ERROR_REPLACEMENT|приложени|прикладанн

# Помилка: «Рабочая среда» (наприклад, операційна система) — це «середовище».
# Чому виникає: Програма сприймає це слово як назву дня тижня.
ERROR_REPLACEMENT|среда|середа

# Помилка: «Загрузка системы» або файлу — це «завантаження».
# Чому виникає: Калька/суржик. Українською можна сказати «грузити» хіба що про мішки в автомобіль.
ERROR_REPLACEMENT|загрузк|загрузк

# Помилка: «Ссылка на сайт» — це «посилання».
# Чому виникає: Суржик. В українській мові «силка»/«ссилка» може асоціюватися хіба що із засланням як видом покарання.
ERROR_REPLACEMENT|ссылк|ссилк

# Помилка: Термін «по умолчанию» має перекладатися як «за замовчуванням» або «стандартно».
# Чому виникає: Машинні перекладачі часто калькують прийменник і пишуть «по замовчуванню» або «по умовчанню».
ERROR_REPLACEMENT|умолчани|замовчуванню

# Помилка: «Портативная колонка» — це «акустична система» або так і залишається «портативна колонка».
# Чому виникає: Перекладач розпізнає слово як елемент електронної таблиці (стовпець/стовпчик).
ERROR_REPLACEMENT|колонк|стовпчик

# Помилка: «Полоса пропускания» у характеристиках мережі — це «смуга пропускання».
# Чому виникає: Калькування російського слова (суржик).
ERROR_REPLACEMENT|полос|полос

# Помилка: «Исключение из правил» — це «виняток» (Винятком із правил є випадки, коли виробник чохла і планшета - один і той же).
# Чому виникає: Перекладач плутає з дією — «виключення» (наприклад, пристрою з розетки, хоча правильно казати «вимкнення»).
ERROR_REPLACEMENT|исключени|виключенн
