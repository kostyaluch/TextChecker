# Приклад таблиці translation_rules.csv

Ось як виглядає файл translation_rules.csv відкритий в Excel або табличному редакторі програми:

| російський_корінь | українські_корені | виключення | тип | коментар |
|-------------------|-------------------|------------|-----|----------|
| питани | харчуванн | збалансоване харчування,дитяче харчування,здорове харчування | Помилка перекладу | Питание (електричне) → живлення, а не харчування |
| лук | цибул | зелена цибуля,ріпчаста цибуля | Помилка контексту | Лук (ігрова зброя) має залишатися «луком», а не «цибулею» |
| разрешени | дозвіл | дозвіл користувача,дозвіл на | Помилка перекладу | Разрешение экрана → роздільна здатність, а не дозвіл |
| отношени | ставленн | ставлення до,дбайливого ставлення | Помилка контексту | Соотношение сторон → співвідношення, а не ставлення |
| шаг | шаг | | Помилка перекладу | Шаг пикселя → крок пікселя (не шаг монета) |
| напряжени | напруженн | напруження для очей,напруження очей | Помилка перекладу | Рабочее напряжение → робоча напруга, а не напруження |
| сеть | сітка | москітна сітка,захисна сітка,металева сітка | Помилка перекладу | Локальная сеть → мережа, а не сітка |
| проводн | проводн | | Суржик | Проводной интерфейс → дротовий (від слова дріт) |
| беспроводн | безпроводн | | Суржик | Беспроводной → бездротовий |
| емкост | ємкіст | | Суржик | Емкость аккумулятора → ємність |

## Переваги такого формату:

1. **Виключення в одному рядку** — не треба шукати в іншому файлі
2. **Категоризація** — можна фільтрувати за типом (Помилка перекладу, Суржик, тощо)
3. **Коментарі** — пояснення чому це помилка допомагає розуміти логіку
4. **Сортування** — можна сортувати за будь-якою колонкою в Excel
5. **Пошук** — легко знайти потрібне правило
6. **Експорт/Імпорт** — можна працювати в Excel і імпортувати назад
