## ✅ Реалізовані зміни

### 1. ✅ Відновлення секції "Режим обробки"

**Проблема:** У версії v17 секція "2. Режими обробки" була повністю видалена, що позбавило користувачів можливості вибирати режим роботи.

**Рішення:**
- Відновлено секцію "2. Режим обробки" з одним чекбоксом
- Чекбокс: "Тільки очищення HTML (без перевірки правил перекладу)"
- За замовчуванням **вимкнено** (дає користувачу можливість вибору)
- Оновлено нумерацію: "Керування та словники" тепер секція 3

**Код:**
```python
# У SpellCheckerApp.__init__()
self.html_only_var = tk.BooleanVar(value=False)  # За замовчуванням вимкнено

# У create_widgets()
options_frame = ttk.LabelFrame(self.root, text="2. Режим обробки")
options_frame.pack(fill='x', padx=10, pady=5)

self.chk_html_only = ttk.Checkbutton(
    options_frame,
    text="Тільки очищення HTML (без перевірки правил перекладу)",
    variable=self.html_only_var
)
self.chk_html_only.pack(side='left', padx=5)
```

### 2. ✅ Поле "Дія" в чорному списку - радіокнопки замість чекбоксу

**Проблема:** У версії v17 поле "дія" було чекбоксом з текстом "Активно", що було незрозуміло:
- ☑ Активно = ?
- ☐ Неактивно = ?

**Рішення:**
- Замінено чекбокс на **радіокнопки (перемикач)**
- Дві чіткі опції:
  - 🔘 **Видаляти** - автоматично видаляє речення
  - 🔘 **Повідомляти** - тільки позначає в звіті

**Код:**
```python
# У RowEditDialog.create_entry_fields()
if col == 'дія' and dictionary_key == 'blacklist':
    var = tk.StringVar(value="Видалити")
    
    radio_frame = ttk.Frame(main_frame)
    radio_frame.grid(row=idx, column=1, sticky='w', pady=5, padx=5)
    
    radio1 = ttk.Radiobutton(radio_frame, text="Видаляти", variable=var, value="Видалити")
    radio1.pack(side='left', padx=(0, 15))
    
    radio2 = ttk.Radiobutton(radio_frame, text="Повідомляти", variable=var, value="Повідомляти")
    radio2.pack(side='left')
    
    self.entries[col] = var
```

**Зворотна сумісність:**
- Старі значення "✓", "Видалити" → "Видалити"
- Старі значення "Підсвітити", "" → "Повідомляти"
- Логіка обробки залишається незмінною (підтримує обидва варіанти)
