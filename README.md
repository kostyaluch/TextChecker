# Аналізатор Перекладу Описів

Інструмент для перевірки якості українських перекладів товарних описів з виявленням типових помилок машинного перекладу та заборонених термінів.

## Функції

### 1. Перевірка помилок перекладу
Програма аналізує українські тексти на наявність типових помилок машинного перекладу з російської мови, таких як:
- "харчування" замість "живлення" (для електроніки)
- "цибуля" замість "лук" (для ігрової зброї)
- "дозвіл" замість "роздільна здатність"
- та інші поширені помилки

**Словник:** `translation_rules.txt`

### 2. Винятки
Програма підтримує список правильних фраз, які не повинні вважатися помилками:
- "збалансоване харчування"
- "здорове харчування"
- "зелена цибуля"
- тощо

**Словник:** `ignore_rules.txt`

### 3. Чорний список заборонених термінів (НОВЕ)
Програма перевіряє українські описи на наявність підозрілих або заборонених термінів:
- Китайські бренди та торгові майданчики (JD.com, Alibaba, AliExpress, Taobao, Tmall, Pinduoduo)
- Посилання та домени (www., http://, .ru)
- Email-шаблони (@gmail., @mail., e-mail:)
- Месенджери (Viber, Telegram, WhatsApp, Skype)
- Підозрілі маркетингові фрази (копія, репліка, аналог, знижка, акція)
- Інші заборонені терміни

**Словник:** `blacklist_terms.txt`

## Використання

1. **Запустіть програму:**
   ```bash
   python "python error search v.6.py"
   ```

2. **Оберіть Excel-файл** з описами товарів

3. **Виберіть колонки:**
   - Стовпчик RU (російською)
   - Стовпчик UA (українською)

4. **Налаштуйте словники** (опціонально):
   - Натисніть "✎ Словник помилок" для редагування правил перекладу
   - Натисніть "🚫 Словник винятків" для редагування виключень
   - Натисніть "⛔ Чорний список" для редагування заборонених термінів

5. **Запустіть перевірку:** натисніть "▶ Почати перевірку"

6. **Перегляньте результати** у створеному файлі `*_checked.xlsx`

## Формат словників

### translation_rules.txt
```
# Формат: російський_корінь = українські_корені_через_кому
питани = харчуванн
лук = цибул
разрешени = дозвіл
```

### ignore_rules.txt
```
# Кожна фраза з нового рядка (українською)
збалансоване харчування
дитяче харчування
здорове харчування
```

### blacklist_terms.txt
```
# Кожна фраза з нового рядка (будь-якою мовою)
www.
JD.com
Alibaba
копія
репліка
100% гарантія
```

## Примітки

- Перевірка чорного списку є регістронезалежною
- Програма шукає терміни як підрядки, тому довші фрази (наприклад, "e-mail:") запобігають помилковим спрацюванням
- Рядки з помилками виділяються жовтим кольором у результуючому файлі
- Усі помилки детально описуються у колонці "Помилки перекладу"

## Типи помилок у результатах

1. **[ПОМИЛКА]** - типова помилка машинного перекладу
2. **[ЗАБОРОНЕНИЙ ТЕРМІН]** - виявлено підозрілий або заборонений термін з чорного списку
