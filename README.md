# Аналізатор Перекладу Описів

Інструмент для перевірки якості українських перекладів товарних описів з виявленням типових помилок машинного перекладу та заборонених термінів.

## Функції

### Єдиний Словник Правил (Unified Rules Catalog)

Програма використовує єдиний файл `rules_catalog.txt` для всіх типів правил та перевірок:

1. **ERROR_REPLACEMENT** - помилки машинного перекладу (RU → UA)
2. **IGNORE** - винятки (правильні фрази, які не є помилками)
3. **AUTO_REMOVE** - автоматичне видалення заборонених термінів
4. **FLAG_ONLY** - позначення термінів у звіті без автоматичного видалення

**Словник:** `rules_catalog.txt`

## Використання

1. **Запустіть програму:**
   ```bash
   python "python error search v.6.py"
   ```

2. **Оберіть Excel-файл** з описами товарів

3. **Виберіть колонки:**
   - Стовпчик RU (російською)
   - Стовпчик UA (українською)

4. **Налаштуйте словник правил** (опціонально):
   - Натисніть "📝 Словник Правил" для редагування єдиного каталогу правил
   - Або використовуйте legacy-кнопки для окремих словників (перехідний період)

5. **Запустіть перевірку:** натисніть "▶ Почати перевірку"

6. **Перегляньте результати** у створеному файлі `*_checked.xlsx`

## Формат словника правил

### rules_catalog.txt

```
# Формат: ТІППРАВИЛА|параметр1|параметр2|...

# 1. Помилки машинного перекладу
ERROR_REPLACEMENT|російський_корінь|українські_корені_через_кому
ERROR_REPLACEMENT|питани|харчуванн
ERROR_REPLACEMENT|лук|цибул

# 2. Винятки (правильні фрази)
IGNORE|фраза_українською
IGNORE|збалансоване харчування
IGNORE|зелена цибуля

# 3. Автоматичне видалення (вирізається з форматованого тексту)
AUTO_REMOVE|термін
AUTO_REMOVE|www.
AUTO_REMOVE|JD.com

# 4. Тільки позначення (фіксується в помилках, але не видаляється)
FLAG_ONLY|термін
FLAG_ONLY|www.
FLAG_ONLY|samsung
```

### Legacy-файли (для зворотної сумісності)

Якщо `rules_catalog.txt` не існує, програма автоматично створить його з наявних файлів:

#### translation_rules.txt
```
# Формат: російський_корінь = українські_корені_через_кому
питани = харчуванн
лук = цибул
разрешени = дозвіл
```

#### ignore_rules.txt
```
# Кожна фраза з нового рядка (українською)
збалансоване харчування
дитяче харчування
здорове харчування
```

#### blacklist_terms.txt
```
# Кожна фраза з нового рядка (будь-якою мовою)
www.
JD.com
Alibaba
копія
репліка
100% гарантія
```

## Примітки

- Перевірка термінів є регістронезалежною
- AUTO_REMOVE: терміни вирізаються з форматованого тексту
- FLAG_ONLY: терміни фіксуються в помилках, але не видаляються
- Програма шукає терміни як підрядки, тому довші фрази (наприклад, "e-mail:") запобігають помилковим спрацюванням
- Рядки з помилками виділяються жовтим кольором у результуючому файлі
- Усі помилки детально описуються у колонці "Помилки перекладу"

## Типи помилок у результатах

1. **[ПОМИЛКА]** - типова помилка машинного перекладу
2. **[ЗАБОРОНЕНИЙ ТЕРМІН]** - виявлено підозрілий або заборонений термін
3. **[МОВА]** - невідповідність мови в колонці
4. **[КОНТЕНТ]** - проблеми з контентом (китайський текст, технічний HTML)
